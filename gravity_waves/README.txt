gw_data/
- Contains Jupyter notebooks for retrieving and plotting GW data.
- Retrieval of the GW data requires the PyCBC library to be installed: https://pycbc.org/

cost_values/
- Contains saved cost function data for use in the VQA simulations.
- These files can be generated from the plotting notebook in gw_data/ or during the VQA simulations.

experiments/
- Examples of python scripts for VQA simulations.

results/
- Saved VQA simulation results.

analysis/
- Jupyter notebooks for analysis of the VQA simulation results.
